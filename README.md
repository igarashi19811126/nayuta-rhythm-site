# assets

このフォルダの写真・アート作品は、新規公式サイト制作の素材として、送付された元画像をそのままコピーして配置しています。

- `art-red.jpg`
- `art-cup.jpg`
- `art-blue.jpg`
- `member-stairs.jpg`
- `member-studio.jpg`
- `live-01.jpg` から `live-10.jpg`

サイト側では画像の色変換、顔補正、AI加工は行っていません。表示サイズに合わせてブラウザ上で切り抜き表示される箇所はあります。

ライブ写真を追加する場合は、このフォルダに画像を入れて `index.html` の `src` を差し替えてください。
